Introducción al Diseño de Sitios Web

Estimados estudiantes sean bienvenidos al OVI Diseños de Sitios Web. Este OVI nos ofrece una iniciación básica en el campo del diseño web utilizando sistemas de control de versiones, tales como GIT y GITHUB con su cliente para escritorio. Además se utilizaran para su desarrollo y diseño los dos lenguajes más usados tales como HTML5 y CSS3.

Objetivos

1. Enseñar a los estudiantes la forma práctica de crear un repositorio remoto con GITHUB y guardar el historial de cambio sobre el código de un proyecto en un sistema de control de versiones como GIT.
2. Conocer la estructura básica de HTML5 y su aplicabilidad a un proyecto de Diseño WEB.
3. Conocer la estructura básica de CSS3 y su aplicabilidad a un proyecto de diseño WEB.

Se espera que al finalizar el recorrido por el OVI, el estudiante pueda construir sitios web por medio de un sistema de control de versiones, así como de tener los conocimientos básicos para poder escribir código en HTML5 y CSS3; De esta manera potenciar las habilidades en el desarrollo de metodologías y herramientas especializadas en línea.
