Introducción al Diseño de Sitios Web

Estimados estudiantes sean bienvenidos al OVI Diseños de Sitios Web. Este OVI nos ofrece una iniciación básica en el campo del diseño web utilizando sistemas de control de versiones, tales como GIT y GITHUB con su cliente para escritorio. Además se utilizarán para su desarrollo y diseño los dos lenguajes más usados tales como HTML5 y CSS3.

Objetivo general

Dar a conocer los componentes básicos y etapas para el diseño y desarrollo de sitios web con HTML5 y CSS3 mediante la definición de dichos conceptos, el diseño de Mockups y el sistema de control de versiones GIT y GITHUB.

Objetivos específicos

1. Dar a conocer el sistema de control de versiones, sus herramientas y ventajas a la hora de diseñar un sitio web.
2. Definición de Mockups y herramientas, así como de ejemplos para el diseño de sitios web profesionales y dinámicos.
3. Dar a conocer HTML5, qué es, para qué se usa y sus estructuras básicas para el diseño de sitios web profesionales junto con ejemplos que expliquen de forma más explícita.
4. Dar a conocer CSS3, qué es, para qué se usa y sus diferentes herramientas para el diseño de sitios web profesionales junto con ejemplos que expliquen de forma más explícita.
